    Python používa .pyc cache súbory!
Vyčisti ich:
bash# Zastav API (Ctrl+C)

# Vyčisti cache
python clear_cache.py

# Vymaž všetky .pyc súbory
find . -type d -name __pycache__ -exec rm -rf {} + 2>/dev/null
find . -name "*.pyc" -delete

# Reštartuj API
uvicorn app.main:app --reload
